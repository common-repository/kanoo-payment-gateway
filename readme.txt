== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of the Kanoo Payment Gateway, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “Kanoo Payment Gateway” and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating, and description. Most importantly, of course, you can install it by simply clicking "Install Now", then "Activate".

== Frequently Asked Questions ==

= What is Authentication Key and User Id? =

Authentication Key and User Id are two required parameters you will receive in your Audience Relationship Manager (ARM) account.

You can create an account by registering at https://arm.kanoopays.com, then access Account -> Payment menu to generate your Authentication Key and User Id.

= Does this support both production mode and sandbox mode for testing? =

Yes it does. Please note that Kanoo have separate environment for production and sandbox. Please contact Kanoo to setup your sandbox account.
